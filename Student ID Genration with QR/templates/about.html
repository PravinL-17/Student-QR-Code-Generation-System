STUDENT QR CODE GENERATOR

For every educational institute an ID card for a student is compulsory. ID card is said to be the summary of any student. It is highly essential for an educational institute to provide ID card to each and every student. So we did this project “Student Identification using QR code” is being created to decrease the effort of human as we know before people use the hand-written ID card. But our ID is the pure automated solution and it will help to generate student ID card easily. It is a special type of software which will be used to make an advanced ID card with QR code.
